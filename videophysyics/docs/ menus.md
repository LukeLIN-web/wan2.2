这是一个独立的 VideoPhy benchmark 评估工作区，用于生成视频并评分，和 PhyT2V/ 是两个独立项目。

  ┌───────────────────────┬──────────────────────────┬─────────────────────────────────────────────────────────────────────────────────────────────────┐
  │       目录/文件       │           内容           │                                              说明                                               │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ videophy_data/        │ videophy2_prompts.csv 等 │ VIDEOPHY2 benchmark 的 600 条 prompt                                                            │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ videophy_outputs/     │ 600 个 mp4，6.0G         │ TI2V-5B 模型生成的完整 benchmark 视频                                                           │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ videophy_outputs_t2v/ │ 133 个 mp4，1.2G         │ T2V-A14B 模型生成的视频（未跑完，只有 133/600）                                                 │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ videophy/             │ 评估模型代码和权重       │ 含 videophy_2_auto/（VideoPhy-2 评分）、videocon_physics/（VideoCon 评分）、VIDEOPHY2/ 推理代码 │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ run_videophy.py       │ TI2V-5B 生成脚本         │ 多线程多 GPU 并行                                                                               │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ run_videophy_t2v.py   │ T2V-A14B 生成脚本        │ multiprocessing 并行                                                                            │
  ├───────────────────────┼──────────────────────────┼─────────────────────────────────────────────────────────────────────────────────────────────────┤
  │ download_videophy2.py │ 下载 benchmark 数据      │ 从 HuggingFace 拉取                                                                             │
  └───────────────────────┴──────────────────────────┴─────────────────────────────────────────────────────────────────────────────────────────────────┘
